# 🎓 Plataforma de Aulas Online

> Plataforma de ensino online desenvolvida com HTML, CSS e JavaScript puro, sem dependências externas.

---

## 📖 Sobre o Projeto

Sistema de gerenciamento de cursos online que oferece uma experiência completa de aprendizado, desde a navegação por catálogos até o acompanhamento de progresso individual.

## 🗂️ Estrutura de Arquivos

```
plataforma-de-aulas/
│
├── 📄 index.html          # Página inicial
├── 📄 cursos.html         # Catálogo de cursos
├── 📄 aula.html           # Visualização de aulas
├── 📄 perfil.html         # Perfil do usuário
│
├── 🎨 styles.css          # Estilos globais
├── ⚙️ functions.js        # Lógica e interatividade
│
├── 📁 Assets/
│   ├── images/            # Imagens e thumbnails
│   ├── videos/            # Conteúdo de vídeo
│   └── icons/             # Ícones da interface
│
└── 📋 README.md
```

## 🚀 Funcionalidades

### 🏠 Página Inicial
- Banner hero com destaque principal
- Vitrine de cursos em destaque
- Navegação por categorias
- Header responsivo com menu de navegação

### 📚 Catálogo de Cursos
- Sistema de busca inteligente
- Filtros por categoria e nível
- Layout em grid responsivo
- Paginação de resultados

### 🎬 Visualização de Aulas
- Player de vídeo integrado
- Barra lateral com módulos do curso
- Indicador de progresso
- Seção de comentários e discussões
- Download de materiais complementares

### 👤 Perfil do Usuário
- Dashboard pessoal
- Cursos em andamento
- Histórico de certificados
- Estatísticas de aprendizado
- Configurações de conta

## 📦 Como Utilizar

### Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/plataforma-de-aulas.git

# Navegue até o diretório
cd plataforma-de-aulas

# Abra o index.html no seu navegador preferido
```

### Desenvolvimento Local

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Navegue pelas páginas usando o menu superior
3. Experimente os filtros e funcionalidades de busca
